# Python自动化测试脚本 - 使用说明

## 项目简介

这是一个学生信息管理系统的Python自动化测试脚本，用于验证学生管理系统的各项功能。

## 项目结构

```
python_tests/
├── test_utils.py       # 工具类（数据生成、数据校验、测试报告）
├── run_tests.py        # 主测试脚本（运行入口）
├── output/             # 输出目录（自动创建）
│   ├── test_report.txt # 测试报告
│   └── test_data.csv   # 测试数据
└── README.md           # 说明文档
```

### 2. 核心模块

#### StudentDataGenerator（测试数据生成器）
- 自动生成随机学生数据
- 支持批量生成
- 支持导出CSV文件

#### StudentValidator（数据校验器）
- 验证ID合法性
- 验证姓名长度
- 验证年龄范围
- 验证成绩范围

#### TestReport（测试报告）
- 记录测试结果
- 计算通过率
- 生成格式化报告
- 支持导出文件

## 安装步骤

### 第一步：安装Python

1. 访问 https://www.python.org/downloads/
2. 下载最新的Python 3.x版本（推荐3.10或以上）
3. 安装时**务必勾选** "Add Python to PATH"
4. 安装完成后，打开命令行验证：
   ```bash
   python --version
   ```

### 第二步：运行测试

1. 打开命令行，进入项目目录：
   ```bash
   cd C:\Users\hp\Desktop\c++student\python_tests
   ```

2. 运行测试脚本：
   ```bash
   python run_tests.py
   ```

3. 查看结果：
   - 控制台会显示测试报告
   - 测试报告保存在 `output/test_report.txt`
   - 测试数据保存在 `output/test_data.csv`
